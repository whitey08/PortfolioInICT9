Lesson 1

In this lesson, we discussed about the proper input of coding in order for them to work in XHTML.


Proper Nesting
  • The order of the bold tag and italicize tag is important in order for them to work.


Proper Tag Closing
  • In XHTML, tags are required to be closed in order for them to work. A slash will also be used in empty tags in order to close them.


Elements in Lowercase Characters
  • Elements must be written in lowercase characters.


One Root Element
  • All elements must be nested within one root element which is the html tag.


Attributes in Lowercase Characters
  • All attributes must be written in lowercase characters.


Quoted Values
  • All values should be quoted.


Forbidden Attribute Minimization
  • Attributes that don't need values are required to have a value equal to its attribute name.


Image Alternate Texts
  • Image alternate texts are mandatory.


Ampersands in URLs
  • It is required to escape the ampersand in the URL in carrying variables. Therefore, we can use the special character entity (&).


Contents Within BLock-Level Elements
  • Contents must be placed within block-level elements. Another good reason to put the span and div tag into practice.
